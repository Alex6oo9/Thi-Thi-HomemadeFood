# Thi Thi Brand Style Guide
## E-commerce Platform for Homemade Burmese Food

---

## Brand Identity

### Brand Essence
**Thi Thi** (meaning "sweet" in Burmese) embodies warmth, trust, authenticity, and home-cooked goodness. The brand connects Burmese expats and food lovers with authentic homemade dishes, creating a safe and welcoming digital marketplace.

### Brand Values
- **Authenticity**: Real homemade Burmese food from passionate cooks
- **Trust**: Safe, reliable payment and delivery
- **Community**: Connecting Burmese diaspora through food
- **Warmth**: Friendly, welcoming, and approachable
- **Simplicity**: Easy to use, no technical barriers

---

## Color Palette

### Primary Colors
- **Burmese Ruby** `#DC2626` (red-600)
  - Primary brand color, inspired by Burmese rubies
  - Use for: CTAs, primary buttons, important highlights
  - Represents: Warmth, passion, Burmese heritage

- **Golden Saffron** `#F59E0B` (amber-500)
  - Secondary brand color, inspired by monk robes and turmeric
  - Use for: Accents, hover states, success indicators
  - Represents: Prosperity, warmth, homemade cooking

### Neutral Colors
- **Deep Charcoal** `#1F2937` (gray-800) - Primary text
- **Medium Gray** `#6B7280` (gray-500) - Secondary text
- **Light Gray** `#F3F4F6` (gray-100) - Backgrounds
- **Border Gray** `#E5E7EB` (gray-200) - Borders, dividers
- **White** `#FFFFFF` - Cards, surfaces

### Semantic Colors
- **Success Green** `#10B981` (emerald-500) - Order confirmed, available items
- **Warning Amber** `#F59E0B` (amber-500) - Low stock, pending actions
- **Error Red** `#EF4444` (red-500) - Errors, sold out, destructive actions
- **Info Blue** `#3B82F6` (blue-500) - Information, links

### Dark Mode Palette (Future)
- Background: `#111827` (gray-900)
- Surface: `#1F2937` (gray-800)
- Text Primary: `#F9FAFB` (gray-50)
- Text Secondary: `#D1D5DB` (gray-300)

## Typography

### Font Family
**Primary**: `'Inter', system-ui, -apple-system, sans-serif`
- Clean, modern, highly legible
- Excellent for both English and Burmese script
- Good weight range for hierarchy

**Fallback for Burmese**: `'Noto Sans Myanmar', 'Pyidaungsu', sans-serif`
- When displaying Burmese language content

### Type Scale
- **H1**: 2.5rem (40px) / Bold (700) / Line-height: 1.2
  - Use for: Hero headlines, page titles
- **H2**: 2rem (32px) / SemiBold (600) / Line-height: 1.3
  - Use for: Section headers
- **H3**: 1.5rem (24px) / SemiBold (600) / Line-height: 1.4
  - Use for: Card titles, subsections
- **H4**: 1.25rem (20px) / Medium (500) / Line-height: 1.5
  - Use for: Component headers
- **Body Large**: 1.125rem (18px) / Regular (400) / Line-height: 1.7
  - Use for: Important descriptions, introductions
- **Body (Default)**: 1rem (16px) / Regular (400) / Line-height: 1.6
  - Use for: Body text, descriptions
- **Body Small**: 0.875rem (14px) / Regular (400) / Line-height: 1.5
  - Use for: Helper text, metadata
- **Caption**: 0.75rem (12px) / Regular (400) / Line-height: 1.4
  - Use for: Timestamps, small labels

### Font Weights
- Regular (400): Body text
- Medium (500): Emphasis, labels
- SemiBold (600): Headings, important UI elements
- Bold (700): Primary headings, strong emphasis

## Spacing System

### Base Unit: 4px (0.25rem)

### Spacing Scale
- `xs`: 4px (0.25rem)
- `sm`: 8px (0.5rem)
- `md`: 12px (0.75rem)
- `lg`: 16px (1rem)
- `xl`: 24px (1.5rem)
- `2xl`: 32px (2rem)
- `3xl`: 48px (3rem)
- `4xl`: 64px (4rem)

### Component Spacing
- **Card Padding**: 24px (xl)
- **Button Padding**: 12px 24px (md, xl)
- **Section Margins**: 48px (3xl)
- **Element Gaps**: 16px (lg)

---

## Border Radius

- **Small**: 6px (0.375rem) - Buttons, inputs, badges
- **Medium**: 12px (0.75rem) - Cards, images, modals
- **Large**: 16px (1rem) - Hero sections, feature cards
- **Full**: 9999px - Pills, avatars

---

## Shadows

- **Small**: `0 1px 2px 0 rgba(0, 0, 0, 0.05)` - Subtle elevation
- **Medium**: `0 4px 6px -1px rgba(0, 0, 0, 0.1)` - Cards, dropdowns
- **Large**: `0 10px 15px -3px rgba(0, 0, 0, 0.1)` - Modals, popovers
- **XL**: `0 20px 25px -5px rgba(0, 0, 0, 0.1)` - Hero cards

---

## Component Styles

### Buttons
- **Primary**: Burmese Ruby background, white text, medium shadow
- **Secondary**: White background, Burmese Ruby border & text
- **Tertiary/Ghost**: Transparent background, gray text, no border
- **Destructive**: Error Red background, white text

### Cards
- Background: White
- Border: 1px solid Border Gray
- Border Radius: Medium (12px)
- Shadow: Small to Medium
- Padding: 24px

### Input Fields
- Border: 1px solid Border Gray
- Border Radius: Small (6px)
- Focus: 2px Burmese Ruby ring
- Padding: 12px 16px
- Helper Text: Body Small, Medium Gray

### Badges/Tags
- Border Radius: Small (6px)
- Padding: 4px 12px
- Font Size: Body Small
- Colors: Semantic colors with 10% opacity backgrounds

## Imagery Guidelines

### Food Photography
- High quality, appetizing food photos
- Natural lighting preferred
- Show portion sizes clearly
- Authentic home-kitchen aesthetic (not overly styled)

### Image Aspect Ratios
- **Product Cards**: 4:3 or 1:1
- **Hero Sections**: 16:9
- **Seller Avatars**: 1:1 (circular)

### Image Treatment
- Border Radius: Medium (12px)
- Subtle shadow on hover
- Lazy loading for performance

## Icons

### Icon Set
Use **Lucide Icons** or **Heroicons** (outlined style)
- Consistent stroke width: 2px
- Size: 16px, 20px, 24px based on context

### Icon Usage
- Shopping Cart: Shopping bag icon
- Orders: Receipt or clipboard
- Profile: User circle
- Search: Magnifying glass
- Payment: Credit card or KBZPay logo
- Delivery: Truck or scooter
- Status: Check circle, clock, x circle

---

## Layout Guidelines

### Grid System
- 12-column responsive grid
- Gutter: 24px
- Max Container Width: 1280px
- Breakpoints:
  - Mobile: < 640px
  - Tablet: 640px - 1024px
  - Desktop: > 1024px

### Navigation
- Top navigation bar: Height 64px
- Logo: Left aligned
- Primary actions: Right aligned
- Mobile: Hamburger menu

### Product Grid
- Desktop: 3-4 columns
- Tablet: 2-3 columns
- Mobile: 1-2 columns
- Gap: 24px

---

## Accessibility

### Contrast Ratios (WCAG AA)
- Normal text: 4.5:1 minimum
- Large text (18px+): 3:1 minimum
- Interactive elements: Clear focus states

### Focus States
- 2px solid ring in Burmese Ruby
- Offset: 2px

### Interactive Elements
- Minimum touch target: 44x44px
- Clear hover states
- Keyboard navigable

---

## Voice & Tone

### Writing Style
- **Friendly & Warm**: "Welcome home" feeling
- **Clear & Simple**: No jargon, easy instructions
- **Conversational**: "Your order", "Let's get started"
- **Trustworthy**: Clear policies, transparent pricing
- **Inclusive**: Welcoming to all Burmese food lovers

### Example Phrases
- ✅ "Fresh homemade Mohinga, just like home"
- ✅ "Order from trusted home cooks"
- ✅ "Your favorite Burmese dishes, delivered"
- ❌ "Premium artisanal culinary experiences" (too formal)
- ❌ "Leverage our platform" (too corporate)

---

## Animation & Transitions

### Timing
- Quick: 150ms - Hovers, small state changes
- Standard: 250ms - Page transitions, modals
- Slow: 350ms - Large animations, page loads

### Easing
- `ease-in-out` for most transitions
- `ease-out` for entrances
- `ease-in` for exits

### Micro-interactions
- Button hover: Scale 1.02, brightness increase
- Card hover: Subtle lift (shadow increase)
- Add to cart: Success feedback animation
- Loading: Smooth spinner or skeleton screens

---

## Responsive Behavior

### Mobile-First Approach
- Design for mobile first, enhance for larger screens
- Touch-friendly: 44px minimum touch targets
- Simplified navigation on mobile
- Stack layouts vertically on small screens

### Breakpoint Strategies
- **Mobile (< 640px)**: Single column, hamburger menu, bottom cart bar
- **Tablet (640px - 1024px)**: Two columns, visible navigation
- **Desktop (> 1024px)**: Multi-column, sidebar navigation, optimal spacing

## Brand Assets

### Logo Usage
- Primary logo: "Thi Thi" wordmark in Burmese Ruby
- Logo clear space: Minimum 16px around logo
- Minimum size: 120px width
- Background: Use on white or light gray only

### Brand Elements
- Burmese pattern accents (subtle, not overwhelming)
- Food illustrations for empty states

## Implementation Notes

### Technology Stack
- **Framework**: React + TypeScript
- **Styling**: TailwindCSS (utility-first)
- **Icons**: Lucide React or Heroicons
- **Animations**: Framer Motion (optional)

### TailwindCSS Custom Config
```javascript
// Extend with custom colors, spacing, shadows
theme: {
  extend: {
    colors: {
      'burmese-ruby': '#DC2626',
      'golden-saffron': '#F59E0B',
    }
  }
}
```
